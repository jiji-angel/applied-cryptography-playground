# Applied Cryptography Playground

This repository contains **practical, Python-based implementations of cryptographic
primitives and protocols**, with a strong emphasis on **correct usage, security pitfalls,
and real-world engineering considerations**.

The goal is not to reimplement cryptography from scratch, but to:
- Demonstrate how cryptographic APIs should (and should not) be used
- Highlight common mistakes that lead to vulnerabilities
- Explain security reasoning behind design and configuration choices
